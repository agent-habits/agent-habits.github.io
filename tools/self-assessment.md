# Agent Habits Self-Evaluation Template

This template helps teams reason about how the Agentic Systems Habits apply to their project.

It is not an audit.
It is not a compliance checklist.

The goal is to support clarity, discussion, and incremental improvement.

## How to use this template

Teams typically use this template:

- During design reviews
- After incidents or near-misses
- When introducing new agent capabilities
- As preparation for sharing a project publicly

Answers do not need to be exhaustive.
Concise, honest descriptions are preferred.
